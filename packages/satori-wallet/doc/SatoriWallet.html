Class: SatoriWallet

SatoriWallet(infuraId, clearCached)

A wrapper for satori wallet functionality. When a user wishes to connect their self-managed wallet to Satori, call connectWallet. Once a wallet is connected their wallet address and token balance can be retrieved. Note: a wallet must be connected before other functions can be used. Wallet connection is cached between sessions; the wallet will be initialised with the cached wallet if a cached provider is detected. Calling disconnectWallet will clear the cached provider. Initialising this with clearCached true will remove the cached provider from memory
